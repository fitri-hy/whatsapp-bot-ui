### <a id="gemini-ai">Gemini AI</a>
Fitur untuk bertanya atau mencari informasi.
- **Format:**  
  `.gemini-ai {query}`
- **Contoh:**  
  `.gemini-ai cuaca hari ini`

### <a id="translate">Translate</a>
Menerjemahkan teks ke bahasa tertentu.
- **Format:**  
  `.translate-{langID} {text}`
- **Contoh:**  
  `.translate-en Apa kabar?`

### <a id="surah">Surah</a>
Mendapatkan informasi tentang surah tertentu dalam Al-Qur'an.
- **Format:**  
  `.surah {NoSurah}`
- **Contoh:**  
  `.surah 1`

### <a id="ayat">Ayat</a>
Mendapatkan ayat tertentu dari surah.
- **Format:**  
  `.ayat {NoSurah:NoAyah}`
- **Contoh:**  
  `.ayat 1:4`

### <a id="wiki-ai">Wiki AI</a>
Mencari informasi menggunakan AI dari Wikipedia.
- **Format:**  
  `.wiki-ai {query}`
- **Contoh:**  
  `.wiki-ai apa itu AI?`

### <a id="wiki-search">Wiki Search</a>
Mencari artikel di Wikipedia.
- **Format:**  
  `.wiki-search {query}`
- **Contoh:**  
  `.wiki-search sejarah Indonesia`

### <a id="wiki-image">Wiki Image</a>
Mencari gambar terkait dengan query di Wikipedia.
- **Format:**  
  `.wiki-img {query}`
- **Contoh:**  
  `.wiki-img Gunung Merapi`

### <a id="country">Country</a>
Mendapatkan informasi tentang negara tertentu.
- **Format:**  
  `.country {name}`
- **Contoh:**  
  `.country Indonesia`

### <a id="seo">SEO</a>
Mendapatkan informasi SEO tentang domain tertentu.
- **Format:**  
  `.seo {domain}`
- **Contoh:**  
  `.seo example.com` 

Dengan panduan ini, Anda dapat dengan mudah mengakses berbagai fitur yang tersedia di bot WhatsApp. Selamat mencoba!
