### Instalasi

```
git clone https://github.com/fitri-hy/whatsapp-bot-ui.git
cd whatsapp-bot-ui
npm install
```

### Menjalankan Server

- Double klik pada file `start.bat`
- Frontend: `http://localhost:3000/`
